About Sleepy Whisper

Project Overview

Sleepy Whisper is a Sleep Disorder Forecast system that leverages advanced machine learning techniques to classify sleep patterns into three categories: Healthy, Insomnia, and Sleep Apnea. Built with Python and Flask, it provides an intuitive web interface for comprehensive diagnostic insights.

Features

ML Prediction

Predict sleep disorders using Gradient Boosting & QDA models with high accuracy.

Data Visualization

Interactive charts analyzing 15,000 sleep health records with Chart.js.

Prediction History

Track all past predictions with detailed input parameters and results.

PDF Reports

Download professional PDF reports with prediction details and recommendations.

Sleep Chatbot

AI-powered chatbot for sleep health tips, disorder info, and wellness advice.

Model Comparison

Side-by-side comparison of GBC and QDA with confusion matrices and metrics.

Machine Learning Models

1. Gradient Boosting Classifier — An ensemble method that sequentially adds decision trees, where each tree corrects errors of the previous one. Achieves ~97% test accuracy.


2. Quadratic Discriminant Analysis — A probabilistic classifier using Gaussian distributions with flexible quadratic decision boundaries. Achieves ~91% test accuracy.

Dataset

The Sleep Health and Lifestyle Dataset contains 15,000 records with 13 features including sleep duration, quality, physical activity, stress, BMI, blood pressure, heart rate, daily steps, occupation, gender, and age.
